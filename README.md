# Correcto — Professional Writing Assistant for Linux

Correcto is a free, open-source, offline writing assistant for Linux inspired by Antidote. It provides bilingual grammar, style, and clarity checks for English and French, with powerful features designed for students, professionals, researchers, and writers who value privacy, precision, and productivity.

Unlike cloud-based tools, Correcto runs 100% offline — your text never leaves your machine.

Correcto — Assistant de Rédaction Professionnel pour Linux

Correcto est un assistant de rédaction gratuit, open-source et hors ligne pour Linux, inspiré d’Antidote. Il propose des vérifications bilingues de grammaire, de style et de clarté pour l’anglais et le français, avec des fonctionnalités puissantes conçues pour les étudiants, professionnels, chercheurs et écrivains qui valorisent la confidentialité, la précision et la productivité.

Contrairement aux outils basés sur le cloud, Correcto fonctionne 100 % hors ligne — votre texte ne quitte jamais votre machine.
